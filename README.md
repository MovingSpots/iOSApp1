Tim Hortons App ☕📱 

Assignment Overview 

This project is a simple iOS SwiftUI application developed as part of the iOS App Development assignment. The app simulates a daily Tim Hortons coffee run where team members can record and save coffee orders for future use. 

The project demonstrates the basic concepts learned from the HIITFit tutorial, including: 

SwiftUI Views 
NavigationStack 
Forms 
Lists 
State Management 
ObservableObject 
UserDefaults Data Persistence 
Timers 
Git and GitHub Integration 
 

Features 

✅ Add team member coffee orders 

✅ Select drink type and size 

✅ Add custom extras (milk, sugar, etc.) 

✅ Save favourite/regular orders 

✅ Delete orders from the list 

✅ Coffee run countdown timer 

✅ Automatically save and load orders using UserDefaults 

✅ SwiftUI Navigation and Forms
